# Interview Code Overlay - 使用说明

## 🚀 快速开始

### 启动服务
双击运行 `启动脚本.bat` 一键启动前后端服务

或手动启动：
```bash
# 后端服务 (在 backend 目录)
cd backend
node server-simple.js

# 前端服务 (在 web 目录)
cd web
npm run dev
```

### 访问应用
- 前端地址: http://localhost:3000
- 后端API: http://localhost:3001

1. **开发**: 双击 `启动脚本.bat` 启动开发服务器
2. **访问**: 浏览器打开 http://localhost:3000
3. **注册**: 使用邮箱注册新账户
4. **配置**: 登录后选择AI模型和配置选项
